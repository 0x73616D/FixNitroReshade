# ⭐ FixNitroReshade.ASI

## ⭕ Activación:
Automática

## ♨️ Función:
Este script soluciona el error que ocurre al activar el nitro de un vehículo, el cual elimina las configuraciones del reshade hasta que se desactive el nitro.

## ⚠️ Dependencias:
- ASI LOADER
- SA-MP R1 (NO PROBADO EN OTRAS VERSIONES)
